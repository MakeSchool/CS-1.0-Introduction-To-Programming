# Updating Lists

In this video, you’ll learn how to update lists using methods that belong to the list data type in Python.

## Check Your Understanding

<FlashCard>
<Prompt>

What would happen if you ran the following lines of code in the Python console? 

```python
cricket_scores = [12.4, 20]
cricket_scores[2] = 20
```

</Prompt>
<Answer>

This code would cause an `IndexError`, because the index you’re trying to access does not exist yet. To add another item to the end of a list in python, we use the append method: `cricket_scores.append(20)`

</Answer>
