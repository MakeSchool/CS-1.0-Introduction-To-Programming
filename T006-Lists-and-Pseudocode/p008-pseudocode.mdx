## Check Your Understanding

<FlashCard>
<Prompt>

What are three reasons to write pseudocode before you start coding in Python?

</Prompt>
<Answer>

Starting with pseudocode has a lot of advantages! You might use it to:

- Communicate what your code will do to others
- Plan out the individual pieces of an application to make sure they will all work together
- Check your logic to prevent potential bugs in your code
