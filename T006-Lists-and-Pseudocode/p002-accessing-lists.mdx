# Accessing Lists

In this video, you’ll learn how to access lists and use the indexing system in Python.

## Check Your Understanding

<FlashCard>
<Prompt>

How could you access the string value `Fekir` in this team roster? 

```Python
roster = [“Messi”, “Fekir”, “Kroos”]
```

</Prompt>
<Answer>

Because we index lists starting from `0` in Python, we can get the string value `Fekir` using the list index `1` like this: `roster[1]`

</Answer>
