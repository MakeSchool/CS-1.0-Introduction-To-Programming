# Snack Break

Python's built-in input function is a powerful way to make your programs more interactive. Try
following these steps to ask your program's user what their name and favorite snack are!

## Steps to Complete

1. Get the user's name with the built-in function `input()` and assign the value to a variable
   called `name`.

2. Get the user's favorite snack and assign it to a variable with a descriptive name.

3. Print out the user's name and favorite snack in a sentence.
