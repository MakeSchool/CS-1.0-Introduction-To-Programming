## Check Your Understanding

<FlashCard>
<Prompt>

What would you add to this line of code in order to store the value that we get as input from the
console to use later in your program?

`input(“Enter your pet’s age: ”)`

</Prompt>
<Answer>

To store the pet age, we need to assign the value to a variable. Like this!

`pet_age = input(“Enter your pet’s age: ”)`.

</Answer>
