<FlashCard>
<Prompt>

What will this code print?

```
speed_limit = 30
car_speed = 22

if car_speed < speed_limit:
    print("That's a safe speed.")
else:
    print("Too fast! Slow down!")
```

</Prompt>
<Answer>

The `car_speed` is less than the `speed_limit`, so it prints "That's a safe speed."

</Answer>
</FlashCard>

<FlashCard>
<Prompt>

What value of car_speed would make this code print `"That's a safe speed."`?

```
speed_limit = 30

if car_speed < speed_limit:
    print("That's a safe speed.")
else:
    print("Too fast! Slow down!")
```

</Prompt>
<Answer>

If car_speed is a number less than 30, the code will print "That's a safe speed."

</Answer>
</FlashCard>
