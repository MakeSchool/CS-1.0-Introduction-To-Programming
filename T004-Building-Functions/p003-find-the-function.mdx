<FlashCard>
<Prompt>

Reading functions will help you learn to write functions.

What does this function do?

```
def greet(name):
    print("Hello, " + name)
```
</Prompt>
<Answer>

The function greets someone by name, printing "Hello," and then the name.

If it was called as `greet("Amir")`, then it would print `"Hello, Amir"`

</Answer>
</FlashCard>

<FlashCard>
<Prompt>

Read this function, then fill in the value of each expression.

```
def triple(n):
    return n * 3
```

`triple(2)`

Result: ?

`triple(100)`

Result: ?

`5 + triple(3)`

Result: ?

</Prompt>
<Answer>

The function returns three times the value passed in.

`triple(2)`

6

`triple(100)`

300

`5 + triple(3)`

14

</Answer>
</FlashCard>
