# Questions: Cement your knowledge of debugging strategies

<FlashCard>
<Prompt>

What are three strategies that you can use to track down and fix bugs in your code?

</Prompt>
<Answer>

* Use your mental model of Python
* Read the error messages carefully
* Use print statements to determine where your code is behaving unexpectedly

</Answer>
</FlashCard>

<MultipleChoice correct={2}>

Professional programmers rarely write code that contains a bug because they have many years of practice.

1. True
2. False

Everyone writes bugs into their code! A bug can come from simple syntax errors, a misunderstanding of the problem, bad documentation, or many other sources. Because code will always be prone to having bugs, it's important to practice our debugging strategies to identify and fix issues early in the software development process.

</MultipleChoice>
