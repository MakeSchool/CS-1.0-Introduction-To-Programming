# Debugging and Files

You’ve probably noticed that sometimes when you write a program - it doesn’t work! In this topic you
can explore strategies for debugging, or fixing any errors that might prevent your code from
behaving like you expect it to.

We’ll also cover reading from and writing to files on your computer so that you can use more complex
data to level up your programs.
