# What is a Variable?

In this video, you’ll learn how to use a computer’s memory with Python to store information in your code.

## Check Your Understanding

<FlashCard>
<Prompt>

What would a good name be for a variable that represents the number of sharks near a beach?

</Prompt>
<Answer>

You want your variable names to be understandable and descriptive. Some good options to represent the number of sharks might be `number_sharks`, `sharks_near_beach`, or `num_beach_sharks`.

</Answer>
