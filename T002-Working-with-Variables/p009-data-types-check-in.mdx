# Questions: How well do you know your Data Types

<FlashCard>
<Prompt>

What is the type of this variable? `boat_oars = 4`

</Prompt>
<Answer>

This value is an **integer** because it is a whole number with no decimals. If we checked the type in the Console with the line `type(boat_oars)` Python would return the type `int`.

</Answer>
<SelfCheck />
</FlashCard>
