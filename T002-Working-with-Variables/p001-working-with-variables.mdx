# Working with Variables

In this course topic, you’ll learn about some of the building blocks of computer science: Variables and Data Types!  

In order to write computer programs, we need to tell the computer to remember certain pieces of information in its memory. Want to track a player’s score in a game? The number of likes on a social media post? An amount of money in a banking app? The maximum speed for an autonomous car? These values can all be stored in memory!

Jump into this topic to learn how to represent information in computer code with variables!
